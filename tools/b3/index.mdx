# b3 (block3) Overview

b3, also known as block3, provides a modern web interface for uploading and downloading schematics and heightmaps. It combines ease of use with powerful features like authentication, permission management, and content organization.

## Features

### Content Management

- **Upload & Download**: Easily upload and download schematics and heightmaps through a user-friendly web interface
- **Organization**: Sort and organize your content with a flexible tagging system
- **Access Control**: Publish schematics with different access levels to control who can view and download your content

### Security & Authentication

- **OAuth2 Integration**: Enterprise-grade authentication support for secure access
- **Permission Management**: Granular permission system to control user access and actions
- **Session Management**: Secure session handling with configurable cookie secrets

### Integration

- **Fast Async WorldEdit Compatibility**: Seamless integration with FastAsyncWorldEdit for direct in-game usage
- **IP Whitelisting**: Configure IP-based access control for automated uploads
- **User Assignment**: Automatically assign ownership of uploaded content to specific users

## Use Cases

- **Server Administrators**: Manage and organize schematics and heightmaps for your Minecraft server
- **Build Teams**: Share and collaborate on builds with controlled access levels
- **Content Creators**: Publish and distribute your creations with proper tagging and organization
